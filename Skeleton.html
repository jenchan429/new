The Skeleton of the Webpage

The skeleton of the structure of the webpage. It is the basic codes that every webpage needs as it tells the webpage

  • what type of html document standard it is to comply with (!DOCTYPE html)
  • what type of coding language to display (HTML)
  • which content to display (every code that is written in the body.../body section
  • where to pull its tools to be used in the html doc (ex. CSS stylesheet, special fonts, links)

I am going to use https://c9.io as my text editor for this demonstation. After selecting your code editor, open up a new file and save it under "index.html." The home page should always be labeled index.html as it will be more visible in the google search engines if you want your website to be found.
