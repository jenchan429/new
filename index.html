Introduction

Welcome to my first website! I started this blog so I can document my coding journey. It has been a long interest of mine to learn to create a website from scratch. The creation process is empowering. However, the learning process requires a lot of effort and motivation. Thanks to the support of the wonderful people at NYPL's Project Code, I have finally created a website! I hope this guide will be able to help you just as it will provide a resource for me. Feel free to browse around or ask questions in the comment section below.

Getting Started

You will need some tools to get started on your journey to create an website. They are

  • A text editor. The two I use are
    • Brackets (http://brackets.io/)
    • Cloud 9 (https://c9.io)
  • A place to "host" your website for public eyes. I use
    • Github (https://github.com/)
  • A place to get some cool free source, non-copyrighted content (such as images)
    • www.pixabay.com
    • www.freepik.com
    • search.creativecommons.org
    • www.google.com (click images labeled for reuse (under search tools)

    Now that you have your tools, let's begin our coding journey in the Skeleton section of the webpage. Buon Voyage!
