Hosting Your Site on Github

Congratulations! You finally finished your website and would love to show the world!

In order to do so, we need to have a website host our site and give it a url. The steps are as follows

  1. Register for Github account
  2. Then create a new repository on Github


  3. Go back to Cloud 9
  4. Clone your workspace
  5. Open up the cloned workspace
  6. Open up the terminal
  7. Enter git branch gh-pages
  8. Press Enter
  9. Enter git checkout gh-pages
  10. Press Enter
  11. Enter git add --all
  12. Press Enter
  13. Enter git commit -m "new changes"
  14. Press Enter
  15. Enter git push
  16. Enter your user name for github
  17. Press Enter
  18. Enter your password (the terminal masks your password)
  19. Press Enter

This will allow you to link all your content to github from cloud 9. To view your website, you would go to http://username.github.io/your workspace or site name/ (ex. http://jenchan429.github.io/website101/)

To update changes to Github, you can repeat steps 11 to 19.

You can finally celebrate! Do a little dance!
